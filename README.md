📖 ゲーム概要

プレイヤーは企業のネットワーク責任者となり、日々襲い来るサイバー攻撃からシステムを守ります。
しかし、ただ守るだけではありません。限られたリソースの中で「自部署のデータ」を優先的に通し、社内での評価（スコア）を競い合う必要があります。

プレイ人数: 1〜4人 (CPU対戦対応)

プレイ時間: 約10分

🛠 ルール

開示フェーズ: そのラウンドで処理すべきパケット（攻撃や業務データ）が公開されます。

構築フェーズ: 手札のカードを使って、パケットフィルタリングのルール（ACL）を行ごとに構築します。

解決フェーズ: 上から順にルールが適用されます。

Permit（許可）: パケットを通します。攻撃ならダメージ、業務データなら得点になります。

Deny（拒否）: パケットを破棄します。何も起きません。
